Recommended test order

  1. Vidking iframe — no setup needed
  2. Custom player — demo streams without API keys
  3. Stream fetch — optional: needs docker compose up -d + TMDB key
  4. TV gate test — deploy via GitHub Pages + TizenBrew

Optional: full stream resolver

  1. Copy .env.example to .env and add TMDB + OpenSubtitles keys
  2. docker compose up -d (port 8787)
  3. Subtitle proxy starts automatically with npm start (port 8788)
